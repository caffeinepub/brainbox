# Specification

## Summary
**Goal:** Build BrainBox, a gamified educational platform with an Energy Meter system, personalized learning paths for Grades 1-12, adaptive quizzes, unlockable mini-games, and optional Olympiad preparation features.

**Planned changes:**
- Create animated welcome screen with onboarding flow for grade, educational system, and stream selection
- Implement Energy Meter gamification system awarding energy for correct answers, completed quizzes, study time, streaks, and achievements
- Build dashboard displaying energy bar, streak counter, subject cards, achievement shelf, leaderboard, and AI-powered weak topic suggestions
- Create learning system with multiple question types (MCQ, fill-in-blanks, match, drag-drop, reasoning, diagram-based) with adaptive difficulty
- Implement mastery tracking with concept strength meters, spaced repetition, and revision reminders
- Add energy spending system to unlock mini-games (Word Rush, Math Blaster, Science Snap, Memory Grid, Quiz Battle Mode), avatar customization, themes, and tournaments
- Build daily quests, weekly missions, Energy Boost Hours, seasonal events, and limited-time challenges
- Create study resources including interactive notes, summary cards, flashcards, and video placeholders
- Implement secure authentication with Internet Identity, parental controls, screen-time limits, break reminders, and progress reports
- Add accessibility features: text-to-speech, adjustable fonts, dyslexia-friendly option, dark/light modes, offline mode, low-data mode
- Create pop-up notification system for Level Up, Concept Mastered, Energy Full, Streak Milestone, Achievement Unlocked, and Study Reminder
- Add supplementary tools: motivational quotes, background music, sound effects, focus mode, exam countdown, timetable planner, homework reminders, goal-setting
- Implement weekly performance analytics with accuracy, time tracking, improvement graphs, weak-topic detection, and AI study suggestions
- Create achievement badge system with Brain Starter, Energy Master, 7-Day Streak Hero, Math Wizard, Science Champ badges
- Add prominent Olympiad preparation option with information section and 5-question diagnostic quiz
- Design modern UI with brain and lightning theme, colorful palette, smooth animations, clean typography, and responsive design
- Ensure strict safety measures: no ads, educational-only content, privacy protection, child-safe data handling, moderated chat, reporting tools

**User-visible outcome:** Students can log in, complete onboarding, access personalized subject-based learning with adaptive quizzes across multiple question types, earn energy through correct answers and study time, unlock mini-games and customization options, track progress with mastery meters and weekly analytics, earn achievement badges, participate in daily/weekly challenges, and optionally prepare for Olympiads with diagnostic assessments—all within a safe, accessible, gamified educational environment.
